PARTE 1

Enunciado do Problema

Considerando os retângulos de área igual a 3 m2, quanto deve ser x, a medida em metros da base do retângulo, para que o perímetro y do retângulo seja o menor possível?


x y
x = y =


INSTRUÇÕES

Importante: o desenvolvimento desta atividade é feito através de vários exercícios que estão disponíveis no formulário de acompanhamento do aluno.

A variável x representa a medida da base do retângulo. Para alterar o valor de x, digite o número desejado no campo “x =”. Neste campo, além de números decimais, você poderá usar números em outros formatos, tais como 1/2, sqrt(5) (raiz quadrada de 5), sen(8) (seno de 8), pi (o número irracional π), etc. Multiplicações podem ser indicadas com um asterisco ou um espaço em branco (por exemplo, 2*pi ou 2 pi para indicar o número 2 π). Use parêntesis para agrupar termos (por exemplo, 3^(1/2) para indicar a raiz quadrada de 3).

Ao pressionar o botão “Calcular!”, o aplicativo fará o seguinte: (1) ele modificará o formato do retângulo para este valor de x e (2) ele calculará o perímetro y correspondente, marcando o par ordenado (x, y) no gráfico e acrescentando uma linha na tabela. Importante: os valores de x e de y são arredondados para 7 casas decimais.

Caso você dê um clique em uma das linhas da tabela, o aplicativo redesenhará o formato do retângulo para o respectivo valor de x. Para reiniciar o programa, pressione a tecla F5!

Dependendo do valor de x que você escolher, talvez seja necessário ajustar a janela de visualização para visualizar algum ponto em particular. Para isto, use os ícones

,        e    

para reduzir (zoom out), ampliar (zoom in) e transladar a janela de visualização, respectivamente. Para ampliar ou reduzir, clique no ícone correspondente e, então, dê cliques sucessivos na janela de visualização. Para efetuar uma translação, clique no ícone correspondente e, então, clique e arraste a janela de visualização. Para mudar a posição do ponto A, ative a ferramenta cujo o ícone é a seta branca

e, então, clique e arraste o ponto A. Importante: o ponto A só pode ser deslocado para posições com coordenadas inteiras.

Nas versões mais atuais do Firefox, do Internet Explorer e do Google Chrome, é possível usar as combinações de teclas indicadas na tabela abaixo para aumentar ou diminuir o tamanho da fonte do texto e o tamanho das figuras. Isto permite configurar estes navegadores para uma leitura mais agradável.
